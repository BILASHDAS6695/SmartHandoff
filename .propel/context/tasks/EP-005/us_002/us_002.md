# US-002: Detect Drug-Drug Interactions with ≥99% Sensitivity and Generate Pharmacist Alerts

> **Epic:** EP-005 — Medication Reconciliation Agent | **Sprint:** 2 | **Points:** 5 | **Priority:** Must Have

## User Story

**As a** clinical pharmacist,
**I want** the system to detect all major drug-drug interactions from the patient's active medication list and alert me in real time for HIGH-severity pairs,
**so that** I can intervene before a dangerous drug combination reaches the patient.

## Acceptance Criteria

- [ ] **AC-1:** For every pair of active medications, agent queries RxNav Interaction API (`interaction.rxnav.nlm.nih.gov/REST/interaction/list`); results cached in Redis with key `drug-interaction:{rxcui1}:{rxcui2}` and 24-hour TTL
- [ ] **AC-2:** OpenFDA Drug Interaction API used as fallback if RxNav returns error or timeout; if both unavailable: `Medication.interaction_severity=UNKNOWN` and reconciliation flagged as incomplete
- [ ] **AC-3:** Severity mapping: RxNav severity ≥700 or "contraindicated" → HIGH; severity 400–699 → MEDIUM; <400 → LOW; HIGH interactions trigger immediate pharmacist alert
- [ ] **AC-4:** High-risk drug classes automatically trigger HIGH alert regardless of API result: anticoagulants (warfarin, heparin, DOACs), insulins, opioids (morphine, oxycodone, fentanyl)
- [ ] **AC-5:** Pharmacist alert delivered via SignalR push to pharmacist dashboard within 60 seconds of detection; alert contains: drug pair, severity, clinical rationale, recommended action
- [ ] **AC-6:** Sensitivity ≥99% verified against a labelled test set of 200 known major drug interactions (NIH/FDA reference interactions)

## Definition of Done

- [ ] Sensitivity ≥99% verified on 200 reference interaction test cases
- [ ] Redis cache hit rate >80% in staging load test (same drug pairs checked repeatedly)
- [ ] Pharmacist alert visible in dashboard within 60 seconds in integration test

## Dependencies

- EP-005 US-001 (Medication lists fetched and stored)
- EP-TECH US-001 (Redis Cloud Memorystore provisioned)
- EP-003 US-003 (SignalR push for alerts)

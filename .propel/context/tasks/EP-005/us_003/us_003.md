# US-003: Detect Duplicate Medications and Flag Missing Chronic Medications

> **Epic:** EP-005 — Medication Reconciliation Agent | **Sprint:** 2 | **Points:** 3 | **Priority:** Must Have

## User Story

**As a** clinical pharmacist,
**I want** the system to automatically flag duplicate prescriptions and highlight chronic maintenance medications missing from the discharge list,
**so that** patients don't receive double doses and don't go home without medications they depend on.

## Acceptance Criteria

- [ ] **AC-1:** Duplicate detection: medications matching on drug name + route + dose within the same list → `Medication.duplicate_flag=true`; duplicate pair surfaced in Medication Review UI with red badge
- [ ] **AC-2:** Cross-list therapeutic duplicate: same drug class (e.g., two ACE inhibitors) in discharge list → flagged as therapeutic duplicate with `interaction_severity=MEDIUM`
- [ ] **AC-3:** Missing chronic medication: pre-admission medication with status CHRONIC (≥3 consecutive months per MedicationStatement) absent from discharge list with no recorded STOPPED order → `Medication.missing_chronic=true`; physician prompted to add stop order or re-prescribe
- [ ] **AC-4:** Missing chronic medications surfaced in pharmacist queue with `WARNING` level (not immediate alert); physician also notified via approval queue badge
- [ ] **AC-5:** Integration test: synthetic patient with 2 duplicate beta-blockers and 1 missing chronic statin → verify `duplicate_flag=true` on both and `missing_chronic=true` on statin

## Definition of Done

- [ ] 95%+ accuracy verified on 15 synthetic duplicate/missing-med test scenarios
- [ ] UI shows duplicate badges in medication three-panel view (EP-005 US-001)

## Dependencies

- EP-005 US-001 (Medication records in DB with list_type and categorisation)

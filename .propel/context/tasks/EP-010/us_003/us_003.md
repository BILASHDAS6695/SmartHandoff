# US-003: Download Discharge Instructions as PDF

> **Epic:** EP-010 — Patient Portal | **Sprint:** 2 | **Points:** 2 | **Priority:** Should Have

## User Story

**As a** patient,
**I want** to download my discharge instructions as a PDF,
**so that** I can share them with a family carer, print them, or keep them for reference without needing internet access.

## Acceptance Criteria

- [ ] **AC-1:** "Download Instructions (PDF)" button visible on portal instructions screen; triggers client-side PDF generation using `jsPDF` library — no server-side request required
- [ ] **AC-2:** PDF includes: hospital name/logo (configurable), patient first name (from patient JWT claims), discharge date, all instruction sections with correct formatting, warning signs, follow-up appointment date/time, care team contact number
- [ ] **AC-3:** PDF filename: `Discharge_Instructions_{discharge_date}.pdf`; auto-opens in browser PDF viewer on desktop; triggers download on mobile
- [ ] **AC-4:** PDF generated in the patient's current portal language (matching the displayed instructions)
- [ ] **AC-5:** PDF does not include: patient MRN, date of birth, or any identifiers beyond first name — minimising PHI in a patient-held document

## Definition of Done

- [ ] PDF generation tested on Chrome desktop (auto-preview), iOS Safari (download), and Android Chrome (download)
- [ ] PHI review: PDF confirmed to contain only approved fields (first name, discharge date, instructions)

## Dependencies

- EP-010 US-002 (Instructions displayed in portal)

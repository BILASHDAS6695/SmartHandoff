## Definition of Done Checklist

### PDF Download (TASK-001)

- [ ] `PdfDownloadService.download()` generates a PDF file and triggers browser save
- [ ] PDF filename follows pattern: `SmartHandoff_Discharge_Instructions_{firstName}_{dischargeDate}.pdf`
- [ ] PDF contains: patient first name, discharge date, hospital name, all five instruction sections with headings
- [ ] Footer on every page: "Generated by SmartHandoff — Always contact your care team with questions."
- [ ] PHI audit: last name, DOB, and MRN are **absent** from the PDF and the `PdfContext` interface
- [ ] `jsPDF` + `jspdf-autotable` added to `package.json` dependencies
- [ ] "Download PDF" button has `aria-label="Download discharge instructions as PDF"`

### Service Worker & Offline Cache (TASK-002)

- [ ] `ngsw-config.json` present with `discharge-instructions-api` data group configured
- [ ] Cache strategy: `freshness` with `maxAge: 30d` and `timeout: 3s`
- [ ] `provideServiceWorker()` registered in `app.config.ts` with `enabled: !isDevMode()`
- [ ] `OfflineCacheService.evictExpiredDischargeCache()` runs on app init via `APP_INITIALIZER`
- [ ] Cache eviction correctly identifies entries with `discharge_date + 30 days < today`
- [ ] Production build (`ng build --configuration=production`) generates `ngsw-worker.js` in `dist/`

### Offline Banner (TASK-003)

- [ ] `NetworkStatusService.isOffline` signal initialises from `navigator.onLine`
- [ ] Signal updates correctly on `online`/`offline` window events
- [ ] Event listeners removed in `ngOnDestroy` (no memory leaks)
- [ ] `OfflineBannerComponent` renders only when `isOffline()` is `true`
- [ ] Banner text: exactly "You're viewing cached instructions"
- [ ] Banner element has `role="status"` and `aria-live="polite"`
- [ ] Banner SCSS: text/background colours pass 4.5:1 WCAG contrast ratio

### PWA Manifest + Install Prompt (TASK-004)

- [ ] `display: "standalone"` set in manifest

- [ ] Icons: 192 × 192 px and 512 × 512 px PNG files present in `assets/icons/`
- [ ] `index.html` links manifest; includes `theme-color`, iOS PWA meta tags, `apple-touch-icon`

- [ ] Lighthouse PWA audit score ≥ 90

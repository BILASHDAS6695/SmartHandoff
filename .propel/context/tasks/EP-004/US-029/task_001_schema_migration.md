# TASK-001: Add `ai_assisted_label`, `approved_at`, and `reviewed_by_user_id` Columns — Alembic Migration

> **Story:** US-029 | **Epic:** EP-004 | **Sprint:** 2 | **Layer:** Backend — Data Model | **Est:** 2 h
> **Status:** Draft | **Date:** 2026-07-16

---

## Context

US-029 Scenario 4 requires three new fields on the `document` table:

| Field | Type | Constraint | Purpose |
|---|---|---|---|
| `ai_assisted_label` | `BOOLEAN` | `NOT NULL DEFAULT FALSE` | Flags AI-generated provenance; permanent — never reset after approval |
| `approved_at` | `TIMESTAMPTZ` | `NULLABLE` | UTC timestamp set on `PATCH …/approve`; NULL until approved |
| `reviewed_by_user_id` | `UUID` | `NULLABLE FK → app_user(id)` | Identity of the approving clinician for the "Approved by" footer |

`DocumentStatus` already contains `APPROVED` from US-028 TASK-001. This task only extends the
`document` table columns; no enum changes are needed here.

All AI-generated documents created by the Documentation Agent (US-025) must have
`ai_assisted_label = TRUE` set at insert time. This task adds the column and backfills
any existing rows that were created by an agent task.

Extends the existing Document model with three US-029 fields:

        Boolean,

### 4. Extend Documentation Agent Insert in `backend/agents/documentation/document_writer.py`

Ensure `ai_assisted_label=True` is set on every `Document` row the agent creates.

```python
async def _persist_document(
    db: AsyncSession,
    encounter_id: UUID,
    content: dict,
    document_type: str,
) -> Document:
    """
    Persist a newly generated document with ai_assisted_label=True.

    US-029 requires the label to be TRUE for every AI-generated document.
    It is set here at the ORM layer so no caller can omit it accidentally.
    """
    doc = Document(
        encounter_id=encounter_id,
        content=content,
        document_type=document_type,
        status=DocumentStatus.PENDING_REVIEW,
        ai_assisted_label=True,          # US-029 — permanent provenance flag
        approved_at=None,
        reviewed_by_user_id=None,
    )
    db.add(doc)
    await db.flush()
    return doc
```

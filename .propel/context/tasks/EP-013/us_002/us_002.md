# US-002: OTP Delivery via Twilio Verify for Patient Portal Authentication

> **Epic:** EP-013 — Notification & Communication Infrastructure | **Sprint:** 2 | **Points:** 3 | **Priority:** Must Have

## User Story

**As a** patient,
**I want** a one-time passcode delivered to my mobile phone to authenticate my portal access,
**so that** I can log in securely without needing a password I would have to remember during a stressful health recovery.

## Acceptance Criteria

- [ ] **AC-1:** `POST /api/v1/auth/patient/otp {portal_token}` → validates portal token signature → calls Twilio Verify `POST /v2/Services/{service_sid}/Verifications` with `{To: patient_phone, Channel: sms}` → Twilio delivers 6-digit OTP SMS within 30 seconds
- [ ] **AC-2:** OTP verification `POST /api/v1/auth/patient/verify {otp, portal_token}` → calls Twilio Verify Check API `POST /v2/Services/{service_sid}/VerificationChecks {Code: otp, To: patient_phone}` → on `status=approved`: issue patient JWT
- [ ] **AC-3:** OTP rate limiting: 5 OTP requests per phone number per hour tracked in Redis; 6th request → 429 Too Many Requests → Twilio Verify NOT called (rate limit enforced before Twilio API call to avoid charges)
- [ ] **AC-4:** Twilio Verify handles OTP hash storage and expiry (10-minute Twilio default); SmartHandoff does NOT store OTP values or hashes — delegates entirely to Twilio Verify service
- [ ] **AC-5:** If patient phone not on file (null in FHIR): portal authentication fails gracefully with "Please contact your care team to update your phone number" message

## Definition of Done

- [ ] OTP delivered and verified in staging with real Twilio Verify credentials
- [ ] Rate limiting verified: 6th OTP request confirmed to NOT reach Twilio (verified via Twilio usage logs)
- [ ] 10-minute expiry tested: enter OTP after 11 minutes → Twilio returns `status=pending` or `expired` → 401 returned

## Dependencies

- EP-TECH US-004 (Twilio Verify service SID in Secret Manager)
- EP-010 US-001 (Patient portal authentication flow calling this endpoint)

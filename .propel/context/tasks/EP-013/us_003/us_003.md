# US-003: Track Notification Delivery Status and Alert on Failures

> **Epic:** EP-013 — Notification & Communication Infrastructure | **Sprint:** 2 | **Points:** 3 | **Priority:** Must Have

## User Story

**As a** care manager,
**I want** to know immediately if a critical notification (portal link, check-in, escalation alert) fails to deliver,
**so that** I can follow up via an alternative channel and ensure patients and staff receive time-sensitive information.

## Acceptance Criteria

- [ ] **AC-1:** `notification_log` table tracks: `notification_id`, `encounter_id`, `channel`, `recipient_masked` (last 4 digits of phone or masked email), `template_name`, `status` (PENDING|SENT|DELIVERED|FAILED|UNDELIVERABLE|OPTED_OUT), `sent_at`, `delivered_at`, `retry_count`
- [ ] **AC-2:** Twilio SMS delivery webhook `POST /webhooks/twilio/status`: on `MessageStatus=delivered` → updates `status=DELIVERED`, sets `delivered_at`; on `MessageStatus=failed|undelivered` → triggers retry logic; after 3 failures → `status=UNDELIVERABLE`
- [ ] **AC-3:** `UNDELIVERABLE` notification for HIGH-priority templates (PATIENT_PORTAL_LINK, HIGH_PRIORITY_CLINICAL_ALERT, PATIENT_ESCALATION_REQUEST) → publishes `NOTIFICATION_FAILURE_ALERT` to Pub/Sub → care team nurse and care manager alerted via SignalR dashboard
- [ ] **AC-4:** Notification history queryable via `GET /api/v1/notifications?encounter_id={id}` (requires NURSE or higher role); shows delivery status per notification with timestamp
- [ ] **AC-5:** Notification delivery rate KPI: `notification_delivery_rate = DELIVERED / (SENT + DELIVERED + FAILED)` tracked in `mv_kpi_daily`; P3 alert if delivery rate falls below 95% over 24-hour window

## Definition of Done

- [ ] Twilio webhook endpoint tested with Twilio webhook simulator: delivered and failed status updates correctly logged
- [ ] UNDELIVERABLE notification for HIGH-priority template confirmed to trigger care team SignalR alert in staging
- [ ] Delivery rate metric confirmed in mv_kpi_daily after 24-hour test window with synthetic data
